## Introduction

Start your journey with Documentation.AI by setting up your environment and creating your first document.

<Callout kind="tip">
Ensure you have an active account to access all features.
</Callout>

## Setting Up Your Environment

<Tabs>
  <Tab title="Windows" icon="monitor">
    ```bash
    winget install documentation-ai
    ```
  </Tab>
  <Tab title="macOS" icon="apple">
    ```bash
    brew install documentation-ai
    ```
  </Tab>
  <Tab title="Linux" icon="terminal">
    ```bash
    sudo apt install documentation-ai
    ```
  </Tab>
</Tabs>

## Creating Your First Document

1. **Log In**: Access your dashboard.
2. **Navigate to Documents**: Click on the 'Documents' tab.
3. **Create New Document**: Select 'New Document' and start writing.

<Expandable title="Read More Tips" default-open="false">
  - Use templates to speed up document creation.
  - Collaborate with your team in real-time.
</Expandable>
